# ULC Brand Tokens

v0.1.1 fixes the minimal brand system for the static ULC LMS prototype.

## Core Tokens

```css
:root {
  --ulc-blue: #3C4AF5;
  --ulc-bg: #f6f7fb;
  --ulc-card-bg: #ffffff;
  --ulc-radius: 8px;
  --ulc-shadow: 0 18px 55px rgba(38, 45, 78, 0.12);
  --ulc-text: #172033;
  --ulc-muted: #6d7486;
}
```

## Product Style

- Main accent color: `#3C4AF5`.
- Light background.
- Rounded buttons.
- Card-based layout.
- Clean modern LMS style.
- Friendly educational tone.
- Russian UI for Beginner students.
- English learning content.
- Desktop-first teacher dashboard.
- Mobile-friendly student homework.

## Usage Rules

- Use `--ulc-blue` for primary actions, active navigation, and key progress moments.
- Use `--ulc-bg` for the app canvas.
- Use `--ulc-card-bg` for panels, cards, dashboards, and exercise surfaces.
- Use `--ulc-radius` for buttons, cards, fields, tags, and status blocks.
- Use `--ulc-shadow` only for meaningful surfaces; avoid decorative noise.
- Use `--ulc-text` and `--ulc-muted` for accessible hierarchy.

## Content Tone

Student instructions stay in Russian and must feel supportive, short, and clear.

Learning examples, dialogues, target phrases, and practice prompts stay in English, with Russian scaffolding where needed for Beginner learners.
