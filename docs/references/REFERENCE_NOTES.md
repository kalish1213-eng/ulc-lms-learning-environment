# Reference Notes

The user shared Student Book reference screenshots in ChatGPT for visual direction.

They are not copied into this repository because ULC project rules explicitly prohibit copying English File / New English File workbook pages, page design, images, characters, texts, exercises, or other copyrighted learning materials.

Use those references only as structural guidance:

- high content density;
- two-column spreads;
- numbered sections;
- visual material embedded in exercises;
- varied short exercise mechanics;
- audio and pronunciation markers;
- useful phrases boxes;
- speaking outcome at the end of the lesson.

All implemented ULC lesson content and imagery in this repository must remain original.
