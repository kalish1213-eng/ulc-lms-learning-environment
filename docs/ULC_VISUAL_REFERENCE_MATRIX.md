# ULC Visual Reference Matrix

| Zone | Main Reference | What We Use | What We Avoid |
|---|---|---|---|
| Student Book | English File structure + ULC brand | Editorial layout, density, photo-task composition, section numbering, G/V/P labels, original ULC content | Copied textbook content, poster-like hero blocks, fake mascots |
| Brand identity | Right ULC case | Colors, typography, logo rules, urban energy, official illustration system | Homemade logos, random SVG, all colors at once |
| Dashboard | ULC brand + professional EdTech shell | Photography, dynamic typography, next action, progress, one main CTA | Generic SaaS card wall, old stick figures, pastel-only hero |
| Unit page | ULC brand | Bold Unit identity, photos, progress, speaking outcomes, section markers | Icons instead of lesson covers |
| Homework | Student Book continuity | Related lesson photo, compact progress, editorial exercises, same section colors | Separate cartoon visual language |
| Review | ULC illustration system | Approved character or asset slot, achievements, scores, mistakes to review | Unapproved mascot, random emoji |
| Teacher Live | Professional workspace | Clarity, control, student list, Student Book center, right-side controls | Promotional poster, decorative brand noise |
