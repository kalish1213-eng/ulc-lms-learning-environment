# ULC Character Reference Bible

Purpose: define original recurring Beginner-course characters inspired by ULC audience, atmosphere, and public visual direction. These are composite fictional characters, not exact likenesses of people from Instagram.

## Continuity Rules

- Face structure must not change between generated frames.
- Hair color, length, and hairstyle remain stable.
- Age impression remains stable.
- Outfit remains stable inside one scene or sequence.
- Accessories do not disappear between consecutive frames.
- Location continuity is preserved inside a sequence.
- No exact likeness of a real person unless `approved_exact_likeness` is explicitly documented.
