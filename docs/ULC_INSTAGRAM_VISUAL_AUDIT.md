# ULC Instagram Visual Audit

Date: 2026-06-20  
Source: https://www.instagram.com/ulc.minsk/  
Scope: public-visible profile, Highlights, 12 visible feed/Reels cards, and public detail frames available without login.

## Общая Визуальная Идентичность

Аккаунт создает ощущение активной городской языковой школы: много движения, разговорных форматов, преподавателей в кадре, промо-механик и коротких образовательных объяснений. Это хорошо совпадает с официальной логикой Right: ULC как Urban Language Club, современный, яркий, разговорный бренд.

Соответствие официальному брендингу сильнее всего видно в цвете и тоне: синий, белый, темно-синий, красный/коралловый акцент, короткие энергичные заголовки, дружелюбная коммуникация. Отличие от типовой языковой школы: меньше академичности, больше живого speaking-club ощущения.

## Люди

В public-visible материалах часто видны преподаватели или преподавательские персонажи, но точная идентичность людей не должна переноситься в LMS без прав. Для production-героев нужны оригинальные составные персонажи.

- Возраст визуально: в основном взрослые 25-40, плюс отдельные youth/teen материалы.
- Одежда: casual-smart, рубашки, кардиганы, свитеры, городская повседневность.
- Эмоции: улыбка, вовлеченность, разговор, легкая импровизация.
- Дистанция: для ULC работают кадры 1:1, small group, teacher-at-board, pair speaking.
- Повторяющийся типаж: открытый преподаватель, взрослый студент, энергичный ведущий объяснения.

## Помещения

По Instagram и официальному сайту подходят следующие направления:

- classroom / office with board;
- reception / service information context;
- group class setting;
- informal language club / cafe mood;
- online/offline hybrid communication.

В LMS не стоит копировать конкретный интерьер один-в-один. Нужно генерировать узнаваемую ULC-атмосферу: светлые помещения, аккуратные рабочие поверхности, взрослые студенты, доска или ноутбук, без детского класса.

## Фотостиль

Подходящий стиль: editorial/documentary, не рекламный stock.

- Ракурсы: eye-level, medium shot, over-the-shoulder, small group wide.
- Крупности: портрет 4:5, сцена 16:9/3:2, sequence 4:3.
- Свет: естественный дневной или мягкий офисный.
- Цвет: чистый светлый фон + ULC blue/navy + один дополнительный акцент.
- Динамика: разговор, жест, письмо на доске, совместная работа.

## Брендовая Графика

Instagram активно использует social-first графику: плотный текст, яркие заголовки, промо, Reels hooks. Это полезно для tone-of-voice, но не должно напрямую переноситься в Student Book.

Что переносить:

- уверенную ULC-синюю основу;
- дружелюбные короткие заголовки;
- смелые акцентные цвета дозированно;
- modern adult language club mood.

Что не переносить:

- перегруженные промо-постеры;
- много текста поверх учебного фото;
- youth-camp визуалы в adult Beginner MVP;
- точные лица людей;
- Instagram UI.

## Сценарии Для LMS

| LMS zone | Подходящие Instagram-сигналы |
|---|---|
| Student Dashboard | speaking-first hero, one clear action, adult learning confidence |
| Unit covers | group/cafe/classroom scene with ULC palette |
| Lesson thumbnails | teacher board, pair speaking, city/cafe detail |
| Student Book | editorial photo + compact task density |
| Reading | real-life profile/contact context |
| Listening | photo sequence with the same characters |
| Speaking | pair work, gesture, eye contact |
| Homework | calm continuation, notebook/phone/laptop |
| Review | achievement/certificate color logic |
| Teacher Live | teacher at board, live responses, small group |
| Onboarding | friendly teacher/student introduction |
| Achievements | certificates and progress, not loud promo |

## Неудачные Решения

- Youth/children visuals are not suitable for Beginner English for Adults.
- Giveaway/promo graphics are too campaign-like for lesson pages.
- Dense schedule cards should not become Student Book layouts.
- Exact teacher/student faces must not be used as production characters.
- Low-resolution thumbnails should not be stretched.
- Social text overlays should not replace interactive LMS content.

## Verdict

Ready for art-direction use. Not ready for production image use. The next production step must generate original, rights-safe ULC images using the character bible and image asset plan.
