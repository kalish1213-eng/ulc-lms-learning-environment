# Content Provenance v0.1.1

This file records how v0.1.1 addresses the v0.1 audit note about original ULC content.

## Status

Static prototype content is treated as original ULC sample content for MVP planning and review. No English File / New English File text, dialogue, exercise, workbook page, image, audio, character, or page design is copied.

## Changes in v0.1.1

- Removed the previous stock-style hero dependency from the product surface.
- Replaced the hero with a native ULC-branded UI block driven by CSS tokens.
- Kept lesson text, examples, exercises, teacher notes, AI explanations, and dashboard copy as original sample material.
- Kept listening and speaking as mock text/contracts only; no external audio source is included.

## Guardrails

- Do not import PDF pages as lesson pages.
- Do not copy published coursebook dialogues or exercise wording.
- Do not use external stock visuals unless an approved asset source is documented.
- Keep ULC-owned lesson structure, examples, and support language in the repository.

## Current Limitation

This is still a static prototype. A full editorial provenance workflow for the complete Beginner course should be added when real content production starts.
