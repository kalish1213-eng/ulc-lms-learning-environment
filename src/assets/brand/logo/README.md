# ULC Logo Assets

Only approved official ULC logo files may be placed in this directory.

Current status: no approved binary logo assets are available in the repository.

Required variants:

- `ulc-logo-round-blue.svg` or approved PNG;
- `ulc-logo-round-white-on-blue.svg` or approved PNG;
- `ulc-logo-round-white-on-red.svg` or approved PNG;
- `ulc-logo-text-blue.svg` or approved PNG;
- `ulc-app-icon.svg` or approved PNG.

Do not add approximate CSS logos, traced recreations, or unapproved redraws.
