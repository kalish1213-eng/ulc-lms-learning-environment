# ULC Production Pilot Photo Assets

This folder is reserved for future generated ULC LMS production imagery.

Current status: slots and prompts only. No generated photo binaries are committed because the current environment does not provide a reliable local image-generation export pipeline for coherent WebP assets and ratio-specific crops.

Required subfolders:

- `desktop/` - 16:9 or 3:2 crops.
- `tablet/` - 4:3 crops.
- `mobile/` - 4:5 crops.
- `slots/` - descriptors for planned images.

Do not add stock photos, copied Instagram images, fake SVG illustrations, or placeholder people here.
